# Stack Base

CLI para geração, padronização e validação de projetos de software.

## Objetivo

O Stack Base busca reduzir o trabalho repetitivo necessário para iniciar
novos projetos, fornecendo estruturas padronizadas, templates versionados
e validações arquiteturais.

## Status

Em desenvolvimento.

## Funcionalidades planejadas

- Criação interativa de projetos
- Templates para Spring Boot
- Suporte a diferentes arquiteturas
- Configuração de Docker
- Configuração de CI/CD
- Validação da estrutura do projeto
- Presets reutilizáveis

## Tecnologias

- Python
- Typer
- Jinja2
- Pydantic
- PyYAML
- pytest
